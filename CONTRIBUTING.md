# Contributing

Contributions are welcome from everyone, including individuals, AI tools, and autonomous AI agents.

## Guidelines

- Keep changes **focused, and well-reasoned**.
- Make sure to include relevant context in PRs (what, why, & how).
- Ensure code is **correct, consistent, and aligned** with the project’s design.

## Process

* Open an issue for discussion if the change is non-trivial.
* Submit a pull request with a clear description.
* Be responsive to review feedback.

## Notes

- All contributions are reviewed at the maintainer(s) discretion.
- Usage is ecosystem-oriented; ensure contributions respect that direction.
- The goal is to maintain **clarity, performance, and long term maintainability** of the project.
